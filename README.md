# Geospatial Workforce Institute Website

Static promotional website rebuilt for three audiences:

1. Individuals starting a surveying career / preparing for CST Level 1
2. Survey companies developing workforce talent
3. Schools building CTE surveying pathways

## Files

- `index.html` — homepage with three entry paths
- `start-career.html` — individual career/CST pathway
- `courses.html` — duplicate of start-career.html for older links
- `for-companies.html` — survey company workforce page
- `for-schools.html` — school/CTE pathway page
- `about.html` — institute mission and purpose
- `contact.html` — contact and audience-specific inquiry guidance
- `styles.css` — shared styling
- `logo.png` — site logo

Upload these files to the root of the public website repository.
